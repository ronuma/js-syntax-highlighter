## Innstructions

1. Clone the repository

2. Navigate to the project directory

3. `$ iex main.exs`

4. `iex> JSSH.run()`

5. This runs the JS syntax highlighter, with "test.js" as an input and "index.html" as an output

6. Open the html in your browser and see the results!
